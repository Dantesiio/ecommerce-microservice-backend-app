# Metodología Ágil Implementada

## Framework: Scrum Simplificado

### Roles
- Product Owner: Santiago
- Scrum Master: David  
- Developer: Santiago y David

### Sprint Duration
1 semana (4 sprints total: 4-25 noviembre)

### Ceremonias
- Sprint Planning: Lunes inicio de semana (30min)
- Daily Standup: Diario (notas personales, 5min)
- Sprint Review: Domingo fin de semana (1h)
- Sprint Retrospective: Domingo (30min)

### Herramienta
GitHub Projects con metodología Kanban

### Definición de Hecho (DoD)
- [ ] Código implementado y funcional
- [ ] Tests pasando (coverage >80%)
- [ ] Documentación actualizada
- [ ] Code review completado (self-review)
- [ ] Desplegado en ambiente dev/local
- [ ] Commit con mensaje descriptivo

## Métricas
- Velocity: Tareas completadas por sprint
- Burn-down: Seguimiento en GitHub Projects
- Quality: SonarQube metrics
